## ğŸš€ Getting Started

### Prerequisites
- **Node.js** 18.0.0 or higher
- **npm** or **yarn** package manager
- **Git** for version control

### Installation

#### 1. Clone the Repository
```bash
git clone https://github.com/yourusername/HYRAX-TASK-MANAGEMENT.git
cd HYRAX-TASK-MANAGEMENT
```

#### 2. Install Frontend Dependencies
```bash
npm install
```

#### 3. Install Backend Dependencies
```bash
cd server
npm install
cd ..
```

#### 4. Environment Configuration

Create a `.env` file in the root directory:
```env
# API Configuration
VITE_API_URL=http://localhost:3001
VITE_WEBHOOK_URL=your_webhook_url_here

# Authentication
VITE_JWT_SECRET=your_jwt_secret_here
```

Create a `.env` file in the `server` directory:
```env
# Server Configuration
PORT=3001
NODE_ENV=development

# JWT Configuration
JWT_SECRET=your_jwt_secret_here

# CORS Configuration
ALLOWED_ORIGINS=http://localhost:5173,http://localhost:3000
```

#### 5. Start Development Servers

**Terminal 1 - Frontend:**
```bash
npm run dev
```

**Terminal 2 - Backend:**
```bash
cd server
npm run dev
```

#### 6. Access the Application
- **Frontend**: http://localhost:5173
- **Backend API**: http://localhost:3001


## ğŸ“ Usage Guide

### Creating Creative Tasks
1. Navigate to the **Tasks** page
2. Click the **"+ Add Task"** button
3. Fill in creative task details:
   - **Title**: e.g., "Instagram Carousel - Spring Campaign"
   - **Description**: Creative brief, copy specs, or asset requirements
   - **Assignee**: Copywriter, designer, or account manager
   - **Deadline**: Campaign launch date or review deadline
   - **Campaign**: Select the client campaign
4. Click **"Create Task"**

### Managing Approval Workflows
1. Click **"+ Add Column"** to create custom approval stages
   - Examples: "Draft", "Copy Review", "Design Review", "Client Approval", "Legal", "Approved", "Live"
2. Drag creative tasks between columns as they progress through approvals
3. Rename columns by clicking the column title to match your agency workflow
4. Delete unused stages using the trash icon (tasks will be preserved)

### Managing Client Campaigns
1. Go to **Campaigns** page to see all client work
2. Browse or search for specific campaigns
3. Click on a campaign to view all associated deliverables
4. Filter tasks by campaign in the Tasks view to focus on specific clients

### Team & Client User Administration
1. Access **User Management** (Admin/Super Admin only)
2. Click **"+ Add User"** to add team members or client stakeholders
3. Assign appropriate roles:
   - **Super Admin/Admin**: Full access for agency leadership
   - **Manager**: Account managers with broad access
   - **User**: Copywriters, designers, team members
   - **Team Member**: Junior staff or freelancers
4. Manage user access and status

### Campaign Deadline Planning
1. Visit **Scheduled Tasks** page for calendar view
2. Use the calendar to select date ranges for campaign planning
